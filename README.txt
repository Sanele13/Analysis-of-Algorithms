Name: Binary and Sequential search(Assignment1)

Description: This program uses two search methods, binary search and sequential search. These methods are used to search for mutual friends, located in a list.
	     it takes a word from a list of names and searches the other list for that name. It then prints out the name, position of the name on the list, 
             the number of comparison before reaching the name, and the time it took it find it. 

Instructions: To compile the program, use the make command on the terminal.
	      To run the program, type 'javac Assignment1' on the terminal.
